# SDSC4070 Homework 2
This is Homework 2 for SDSC4070. This homework is to design and implement an experiment where we fine-tune a chosen base LLM using two methods: full finetuning and one PEFT.

The outcome is based on Hiyouga's [LLAMA-Factory](https://github.com/hiyouga/LLaMA-Factory.git).

The two directories retore the logs for Full Finetuning and PEFT accordingly.
